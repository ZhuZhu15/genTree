Выполните команду php gen_tree.php /path/to/input.csv /path/to/output.json

/path/to/input.csv - путь к входному файлу в формате CSV.
/path/to/output.json - путь к файлу, в который будет записана иерархия дерева в формате JSON.

Примечание: Специально не поместил vendor в .gitignore, чтобы пользователь не устанавливать зависимости,
мало ли composer'a нет =)
